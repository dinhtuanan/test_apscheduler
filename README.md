# Test APSchedule python library
1. Python version: 3.9.13
1. Virtual Env: venv
## Create venv project
```
python -m venv apschedule_env
```
## Activate venv
```
apschedule_env\Scripts\activate
```

## Install python library
```
pip install -r requirements
```

## Run schedule task
```
python task_apscheduler.py
```
